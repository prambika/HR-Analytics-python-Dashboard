# HR-Analytics-python-Dashboard

HR Analytics Dashboard using Python. This project analyzes employee data including attrition, performance, salary, and attendance using Pandas, Matplotlib, and Seaborn.

---

## 📊 Dashboard Features

* Employee Attrition Analysis
* Salary Distribution
* Performance Insights
* Attendance Tracking

---

## 🛠️ Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn

---

## 📁 Files Included

* app.py → Main dashboard code
* HR_Analytics_Extended_Data.xlsx
* HR_Multi_Sheet_Data.xlsx

---

## ▶️ How to Run

1. Install Python
2. Install libraries:
   pip install pandas matplotlib seaborn
3. Run:
   python app.py
